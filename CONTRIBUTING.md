# Contributing

Thanks for your interest in contributing to the RedRanks SEO Intelligence API examples!

## Ways to Contribute

### Add Examples

We welcome examples in additional languages. If you'd like to add an example:

1. Create a new folder under `examples/` with the language name
2. Include working examples for all three endpoints:
   - `/keyword-metrics`
   - `/serp-analysis`
   - `/discover-threads`
3. Follow the patterns in existing examples
4. Add tutorial links in comments
5. Submit a pull request

### Report Issues

If you find a bug in the examples or documentation:

1. Check if the issue already exists
2. Open a new issue with a clear description
3. Include steps to reproduce if applicable

### Improve Documentation

Documentation improvements are always welcome:

1. Fix typos or unclear wording
2. Add missing information
3. Improve code comments
4. Add more use case examples

## Code Style

- Keep examples simple and focused
- Add comments explaining what the code does
- Use meaningful variable names
- Include tutorial links where relevant
- Test your examples before submitting

## API Issues

For issues with the API itself (not the examples), please contact us through:

- [Website](https://www.redranks.com)
- [RapidAPI](https://rapidapi.com/greg01060/api/reddit-traffic-and-intelligence-api)
